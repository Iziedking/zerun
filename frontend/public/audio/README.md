# Arena soundtrack

Drop a looping background track here as `zerun-theme.mp3` and the music toggle in
the top bar turns on. Without a file the toggle hides itself, so nothing breaks.

Use a royalty-free or CC0 track so the public repo stays clean on licensing. Good
fits for the cosmic AI-arena vibe (chiptune, synthwave, lo-fi electronic):

- Pixabay Music (free, no attribution required): https://pixabay.com/music/search/synthwave/
- Free Music Archive (filter to CC0 / CC-BY): https://freemusicarchive.org/
- Incompetech by Kevin MacLeod (CC-BY, credit in the README): https://incompetech.com/music/royalty-free/

Pick a loop-friendly track, save it as `public/audio/zerun-theme.mp3`, and it
plays at a low volume on a loop. To use a different path or a hosted URL, set
`NEXT_PUBLIC_MUSIC_URL` instead.
